# PWRL Documentation Bundle

The PWRL knowledge bundle contains agent-generated documentation for the PWRL coding agent harness — skills, plans, tasks, learnings, and guides produced by the pwrl-* skill pipeline.

## Directory Index

* [Learnings](learnings/) — Patterns, decisions, workflows, gotchas, technical fixes, and plans extracted from development sessions. Organized by type into decision/, pattern/, workflow/, gotcha/, technical-fix/, and plan/ subdirectories.
* [Tasks](tasks/) — Granular implementation task files sliced from plans. Organized by status into to-do/, done/, and archived/ subdirectories.
* [Analysis](analysis/) — In-depth technical analysis documents covering PWRL skill architecture, GitHub integration design, and comprehensive audit reports.
* [Guides](guides/) — Practical guides for PWRL development: architecture refactoring, micro-skill composition patterns, skill documentation updates, and migration checklists.
* [Examples](examples/) — Usage examples demonstrating PWRL agent workflows, planner patterns, and GitHub integration setups.
* [Plans](plans/) — Structured implementation plans with tiered design (Fast/Standard/Deep), technical decisions, and implementation unit decomposition.
* [Test Plans](test-plans/) — Test reports and test planning documents for PWRL skill validation.
